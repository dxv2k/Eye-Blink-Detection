# Usage: 
Tutorial to install [dlib](https://www.pyimagesearch.com/2017/03/27/how-to-install-dlib/)  

## Install dependencies 
`
    pip install -r requirements.txt
`

## Compile & Run: 
Only available testing version right now:   
` 
    python tests/predictor.py
`
